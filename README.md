PartsMart – Hardware Accessories E-Commerce Platform
📌 Project Overview

PartsMart is an e-commerce platform designed for selling hardware accessories directly from manufacturers to wholesalers. The system provides an easy way for customers to browse products, place bulk orders, and manage purchases through a streamlined online platform.

The platform includes features for customers, administrators, and delivery agents, ensuring smooth product management, order processing, and delivery tracking.
